# 简介

这是一个用于应对运维工作的模块，能够将提前设定好的运维任务流水线式执行，并且支持流程化编排。

<br>

## 特性

- 支持流程状态管理。
- 每个流程/块支持接收上一流程/块的返回值。
- 支持分支流程，需要自己设定判断表达式。
- 支持流程编排。

<br>

## 版本
python3.6+

<br>
